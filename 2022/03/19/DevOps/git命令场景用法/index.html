前言

现在，git已经成为了大家的代码仓库管理的一个工具了。在日常工作，我们会遇到各种个样的git问题，因此，用一篇文章来累计记录，日常生活中，我们会遇到，但是不常用的命令。

概念

其实 git 的概念，我们开发者应该很多有会了，存在以下几个区域：

  • 工作区 （你的任何改动）
  • 暂存区 （git add）
  • 本地仓库 （git commit）
  • 远端代码仓库 （git push）

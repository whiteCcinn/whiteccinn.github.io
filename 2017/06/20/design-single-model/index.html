设计模式の创建型の单例模式

2017-06-20

简介

很容易理解，也很简单。

最常见的场景就是一个数据库的链接，我们每次请求只需要连接一次，也就是说如果我们用类来写的话，只需要用一个实例就够了（多了浪费）。

说明

单例模式其实分为 2 种，PHP 中最常用的是懒汉模式，意识就是需要加载的时候才去实例化，还有一种就是叫饿汉模式，就是一开始就开始实例化了，但是由于 PHP 不支持在类定义时给类的成员变量赋予非基本类型的值。如表达式，new 操作等等，所以 PHP 中就不存在饿汉模式了。

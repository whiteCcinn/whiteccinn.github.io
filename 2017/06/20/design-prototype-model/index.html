2017-06-20

实质就是 对象的复制

对一些大型对象，每次去 new，初始化开销很大，这个时候我们 先 new 一个模版对象，然后其他实例都去 clone 这个模版， 这样可以节约不少性能。
这个所谓的模版，就是原型（Prototype）；
当然，原型模式比单纯的 Clone 要稍微升级一下。

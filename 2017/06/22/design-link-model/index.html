2017-06-22

一个常见的非正统的设计模式

fluent interface（流利接口）有一个更广为人知的名字『链式操作』，可能大多数人大概都是从 Jquery 最先熟悉的，在 laravel 中，ORM 的一系列 sql 操作，也是链式操作，特点是每次都返回一个 Query Builder 对象。

实例

链式操作的关键在于，每次都返回本对象 return $this，使得没一次操作之后都是可以可调用方法的对象。
